# React Package Boilerplate

Using bun, ts and storybook

To install dependencies:

```bash
bun install
```

To build:

```bash
bun run build
```

To run:

```bash
bun run storybook
```

This project was created using `bun init` in bun v1.0.11. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
